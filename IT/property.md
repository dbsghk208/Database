## property

```jsx
  var person = {
	name : "Jason",
	age : 25,
	occupation : "Student",
	getPersonProfile : function() {
		return "Name : " + this.name +
				"\nAge : " + this.age +
				"\nOccupation : " + this.Occupation;

	}
};
```
<br><br>

name age occupation  >> property
getPersonProfile(위의 함수를 담고 있음)
<br>

프로퍼티는 object 를 위해서 데이터를 저장한다.
메소드는 object 가 요청 받을 수 있는 액션
