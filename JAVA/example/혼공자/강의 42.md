 
<핵심포인트>

 기본 타입과 마찬가지로 클래스도 타입 변환이 있음  
 이를 활용하면 객체 지향 프로그래밍의 다형성을 구현할 수 있다.
 
<br>
 
 <다형성>
 
 - 다향한 객체활용해 여러 실행 결과가 나오도록 하는 성질
 - 메소드 재정의와 타입 변환으로 구현
 
 메소드 재정의 + 타입변환 => 다형성
 
<br>
 
 자동타입변환
 
 - 프로그램 실행 도중 자동으로 타입 변환 일어나는 것  
 - 바로 위 부모가 아니더라도 상속계층에서 상위 타입인 경우 자동 타입 변환 일어날 수 있음
