## 매개변수 다형성

생성자나 메소드에서 외부로부터 받는 값을 저장하기 위해 사용하는 변수

매개변수 타입을 부모 타입으로 선언했을 경우 나타나는 효과
- 메소드 호출시 매개값으로 부모 객체 및 모든 자식 객체를 제공할 수 있음
- 자식의 재정의된 메소드가 호출 -> 다형성
